# Transcom LMS Web

Next.js frontend application for the Transcom Learning Management System (LMS), built using TypeScript, Tailwind CSS, and the Next.js App Router.

## Prerequisites

- [Node.js v18.17.0+](https://nodejs.org/)
- npm or another package manager (pnpm, yarn)

## Local Setup

1. Copy the example environment file to create your local environment configuration:
   ```bash
   cp .env.example .env.local
   ```
2. Open `.env.local` and configure your API URL, database connections, and secrets.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
